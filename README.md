# React 시작하기

## Setup

```bash
# https://github.com/coreybutler/nvm-windows/releases
# 여기에서 최신의 nvm(node version manager) 설치

nvm install node 8.10.0(가장 많이 쓰이는 버전. 다른 이와의 호환성을 위함)
nvm use 8.10.0
npm install -g yarn
yarn global add react-create-app
react-create-app [프로젝트 이름]
cd [프로젝트 이름]
yarn start

```
